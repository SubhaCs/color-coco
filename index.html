Exploring Color Information as Auxiliary Conditioning for Image Colourization in MS COCO Dataset

Colourizing grayscale images with various objects and colours poses a significant challenge for many image colourization algorithms, particularly when confronted with intricate real-world scenes. The conventional approaches often encounter difficulties in generating consistent colour outputs due to the one-to-many association of colourization. In this work, we propose a novel methodology that leverages textual descriptions of the objects present in a scene as an additional input to grayscale images, thereby enhancing the accuracy of the colourization process. Our approach involves a deep network that takes two inputs: the grayscale image itself and its corresponding eencoded text description and aims to colourize the grayscale image. We predict the relevant colour components by training the network to understand the correlation between the image and its textual description. To accomplish this, we adopt a multi-modal feature attention approach that helps to colourize the detected objects using the instance object colourization (IOC) module.The proposed IOC module is conditioned on the grayscale image and its associated language description. Our approach diverges from traditional colourization techniques that primarily rely on unconditional generation, where the colours are solely predicted from the input grayscale image. Instead, our approach capitalizes on the fusion of the image and its accompanying text description to achieve more accurate and consistent colourization results. In this paper, we also create a dataset using the COCO dataset that describes the colour information of every object of the dataset.
